# ArenaFlow 2026 🏟️

### FIFA World Cup 2026 - Stadium Operations & Fan Experience Command Hub

ArenaFlow 2026 is a premium, GenAI-enabled stadium operations and tournament navigation suite built for **MetLife Stadium** during the **FIFA World Cup 2026**. Merging fan convenience widgets with operational mission control analytics, this dashboard optimizes crowd movement, transit access, sustainability goals, and emergency protocols in a sleek, high-contrast, glassmorphic layout.

## 🧠 2. Technical Approach & Logical Architecture

### 🧭 Breadth-First Search (BFS) Graph Pathfinder
* **Concept**: The stadium concourse is modeled as a coordinate network graph consisting of seating sections (101–112), gates (A–D), first aid hubs, transit terminals, and concession nodes.
* **Logic**: When a node is selected, a BFS traversal calculates the shortest route between the start pin and destination. It renders an animated vector path (`stroke-dasharray` dashes flowing towards the destination) on the SVG map and updates directions dynamically.

### 🤖 Google Gemini AI Core Config & Telemetry Hub
* **Active Mode**: Connects directly to the Google Gemini API using secure, key-authorized requests.
* **Expert/Standard Mode**: Allows switching reasoning engines between `gemini-2.5-flash` (Standard) and `gemini-2.5-pro` (Expert).
* **AI Directives Editor**: A live settings panel lets judges customize system prompt instructions on the fly, immediately re-prompting the assistant.
* **Real-time Telemetry Monitor**: Tracks and displays AI response speed (latency), loaded context token size, and generated output tokens for every query.
* **Offline Fallback**: If no key is set or the network fails, it falls back to a multilingual search matching keyword arrays in [src/stadiumData.js](file:///d:/promptwars/src/stadiumData.js).

### 💚 GreenFan Eco-Scorecard & Eco-Transit Board
* **Concept**: Incentivizes fans to reduce their carbon footprint.
* **Logic**: Routing from public transit hubs (Gate A MetLife Rail, Gate B Electric Shuttle, Gate C Express Bus) or ordering from sustainable concessions (Vegan Goal, Taco Cantina, Sweet Stadium) automatically rewards fans with Eco Points/XP and carbon offsets, unlocking discounts once threshold badges are met.

### 🛡️ Security & XSS Sanitization
* **Escaped Output Pipeline**: Employs rigorous HTML escaping on inputs before rendering them inside chat boxes or volunteer broadcast logs, neutralising XSS attacks and injection payloads.

### ⚡ Resource Efficiency & Caching
* **Prompt Caching (`sessionStorage`)**: Prompt answers are logged in client-side storage. Recurring queries in the same language fetch instantly from cache, saving Gemini API token quotas and reducing processing latency to 0ms.

## 🎈 6. Streamlit Deployment & Integration
To serve and host this dashboard inside a Streamlit web container:
1. **Prerequisites**: Install Streamlit in your Python environment:
   ```bash
   pip install streamlit
   ```
2. **Build Assets**: Run the production build to compile Vite scripts:
   ```bash
   npm run build
   ```
3. **Run Locally**: Launch the Streamlit server wrapper:
   ```bash
   streamlit run app.py
   ```
4. **Deploying to Streamlit Cloud**:
   * Commit the compiled `dist/` build directory to your public GitHub repository (since Streamlit Cloud runs python-only by default and requires the pre-compiled assets).
   * Ensure `dist/` is NOT blocked by `.gitignore` when pushing if deploying directly from the cloud.
